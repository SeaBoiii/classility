# RPG Result Card

Premium collectible fantasy card UI for a single sample RPG personality result.

## Run

```bash
npm install
npm run dev
```

Open:

- `/card`

## Export PNG

```bash
npm run export
```

Output:

- `out/card.png`

Export details:

- viewport: `900x1400`
- format: `png`
- `omitBackground: true` for transparent outside-card areas
